# Kali's Arena  

A multiplayer browser-based arena game where players compete in a dynamic hexagonal battlefield. Players navigate through strategically generated platforms, gaps, and various elemental terrains.  

## Features  

- **Real-time multiplayer gameplay**  
- **Procedurally generated hex-based maps**  
- **Dynamic platform system** with edges and gaps  
- **Elemental terrain types** affecting gameplay  
- **Smooth player movement and collision detection**  
- **WebSocket-based player synchronization**
